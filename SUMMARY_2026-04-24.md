## Architecture Design (v0.3)

### Four-Tier Memory Model

| Tier | Location | Audience | Lifespan | Promotion |
|------|----------|----------|----------|-----------|
| **Tier 0** | Project-local memory/ | Individual/project | 30 days hot | Auto-extract to drafts |
| **Tier 1** | Project MEMORY.md, agent MEMORY.md | Project + Toan | Permanent | Human-reviewed |
| **Tier 2** | `/srv/openclaw_projects/standards/` | Team (all members) | Permanent | Gated: 2+ projects OR universal, specific, with source+rationale+scope+confidence+reviewer |
| **Tier 3** | Project `.openclaw/` | Project + extraction | Project lifespan | Candidate for Tier 2 (manual review) |
| **Tier 4** | MemoryGraph (execution engine) | Team | Permanent | Extraction → drafts → promotion pipeline |

### Core Principles
1. Authoritative source + derived knowledge (no accidental duplication)
2. Private by default, shared deliberately
3. Promotion is gated (auto-extract to drafts; human approves promotion)
4. Redaction before persistence (strip secrets, log metadata only)
5. Compaction has rollback safety (pre-snapshot, diff review, approval)
6. Measurable multi-signal improvement (rework + clarifying-Qs + round-trips + corrections + contradictions)

### Success Metrics (Month 2)
- Extraction signal/noise ratio > 0.6 approved
- Promotion rejection rate 30-50% (gate working)
- Duplication detections < 10%
- Context round-trips reduction -20% (by M6)
- Redaction false-negatives = 0
