## 5. ImageForge3D Feature Brainstorm: Drawing & Masking 🎨 ✅ CREATED

**Headline Feature:** In-canvas drawing + AI inpainting with seamless blending

### Core MVP (Week 7 after hardening)
**What user does:**
1. Generate initial render
2. Load in UI
3. Draw mask over areas to enhance
4. AI inpaints masked regions only
5. Seamlessly blend with original
6. Export result

**Drawing Tools:**
- Brush (variable size, opacity)
- Eraser (soft/hard)
- Rectangle select
- Freeform lasso
- Polygon select

**Inpainting Backend:**
- Stable Diffusion inpainting pipeline
- Poisson blending for seamless transitions
- Real-time low-res preview
- Support for custom prompts per region

### Advanced Features (Phase 2)
- Mask library (save + reuse across images)
- Multi-layer masks (stack operations: face → background → effects)
- Smart edge detection (auto-feather boundaries)
- Batch inpainting (same mask on multiple images)
- Blender addon (paint masks directly in Blender)
- Upscaling (enlarge masked areas with quality)
- Style transfer (apply styles to masked regions)
- Object removal (smart inpainting)
- Object duplication (place realistic copy)

### Timeline & Complexity
- MVP: 2-3 weeks (Week 7 of hardening schedule)
- Advanced: Week 8+
- Tech needed: Fabric.js (drawing), Pillow (mask ops), HF Diffusers (inpainting)

### Open Questions
1. Which drawing tools are must-have? (all or subset?)
2. Blending algorithm: Simple alpha vs. Poisson?
3. Real-time preview or "Generate" button only?
4. Support multiple masks at once?
