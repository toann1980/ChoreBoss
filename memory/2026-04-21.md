## Key Insights This Session

### 1. Docker Isolation is Security
- Containers can't reach external DNS by default (intentional)
- Pre-compiled builds are the right approach for restricted networks
- Global DNS config cleaner than per-service overrides

### 2. Design Decisions Matter
- Cascade fallback ensures signals always generate (even if LLM fails)
- PostgreSQL vanilla is simpler than TimescaleDB for 3-month datasets
- Port registry prevents conflicts across 5 projects
